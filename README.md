#### Get all prefectures

```http
  GET /search/repositories/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `q` | `string` | **Required**. search key |
| `sort` | `string` | **Optional** stars ,forks,..|
| `order` | `string` | **Optional** desc or asc |
